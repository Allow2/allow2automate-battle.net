# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.2] - 2024-01-15

### Added
- Initial public release
- Enable/disable Battle.net access control
- Real-time quota checking with Allow2 API
- Automatic quota update functionality
- Quota exceeded event trigger
- Quota renewed event trigger
- Support for multiple child profiles
- Network and configuration permissions

### Changed
- Improved error handling for network requests
- Enhanced quota synchronization logic

### Fixed
- Connection stability with Battle.net servers
- Quota calculation edge cases

## [0.0.1] - 2024-01-01

### Added
- Initial development version
- Basic plugin structure
- Core integration with Allow2Automate
- Battle.net API integration
- Basic quota management
